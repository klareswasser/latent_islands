# 潜在島マップ / Latent Islands

海・河川・運河によって分離した、日本全国の「潜在的な島」を可視化するWeb地図です。

**[Web地図を開く](https://klareswasser.github.io/latent_islands/)**

## 定義

国土数値情報 N03 の各天然陸塊から、W05 河川線に分析用の幅を与えた水域を差し引きます。切断後の最大成分は元の本土・島として除外し、新たに分離した成分のみを「潜在島」とします。

そのため、元から海で囲まれた通常の島そのものは集計対象に含みません。橋梁は陸続きとみなしません。

## データと分析条件

- 行政区域 N03：2025年版、47都道府県
- 河川 W05：2006〜2009年度版、47都道府県
- 採用河川区分：1（一級直轄）・2（一級指定）・3（二級）
- 分析用半幅：45 m・30 m・18 m（実測河川幅ではありません）
- 表示下限：0.02 km²
- 抽出結果：626島
- 分析対象：W05 97,697区間
- 隣接色の判定距離：500 m

W05 は水域面ではなく河川中心線です。河川幅の仮定により結果が変化するため、本地図は法的・地名上の島を示すものではなく、上記の分析定義に基づく実験的な可視化です。

## 再生成

Python 3 と `geopandas`, `shapely`, `pandas`, `numpy` が必要です。国土数値情報のデータを `data/extracted/N03` と `data/extracted/W05` に展開し、以下を実行します。

```bash
python3 process.py
```

主要パラメータは `process.py` の `HALF_WIDTH_M`, `MIN_AREA_KM2`, `NEIGHBOR_DISTANCE_M` で変更できます。
